# LocalQuery: Ask Your Docs

**LocalQuery** is a hackathon project that lets you run a powerful Question-Answering AI on your own documents, 100% on your laptop.

No data ever leaves your machine. Upload a PDF, .docx, or .txt file and ask it questions.

## Features

  * **Local-First:** Runs entirely on your laptop.
  * **Secure:** Your documents are never uploaded to any server.
  * **File Support:** Works with `.pdf`, `.docx`, and `.txt` files.
  * **Powered by Transformers:** Uses a pre-trained `distilbert-base-cased-distilled-squad` model for fast and accurate answers.
  * **(Stretch Goal) Fine-Tuned Model:** Includes a fine-tuned model for enhanced performance on [Your Domain] documents.
